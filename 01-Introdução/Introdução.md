## Banco de Dados II

Componente Curricular: Banco de Dados II  
Semestre: 3º  
Carga horária: 80h (67h relógio)
Professor: Alcione Benacchio.

---
## Objetivo

Aprofundar o uso de SGBDs relacionais, focando em:

- Gerenciamento de transações    
- Controle de concorrência
- Recuperação de falhas
- Otimização de consultas

---

## Ementa

* Paradigmas de armazenamento de dados  
* Otimização de consultas  
* Indexação  
* Gerenciamento de transações  
* Controle da concorrência  
* Stored Procedures e Functions  
* Triggers  
* Administração de banco de dados  
* Tópicos especiais em banco de dados
